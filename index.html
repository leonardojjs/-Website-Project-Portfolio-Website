My academic background combines engineering mathematics, numerical methods, and computational modeling with extensive hands on experience using Python for data analysis, simulation, and applied artificial intelligence. As a coding teacher, I focus on building strong logical thinking, problem solving skills, and practical understanding, helping students not only write code, but also understand why the code works and how to apply it to real world problems.
